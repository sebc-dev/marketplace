# sebc.dev marketplace

Plugin marketplace for [Claude Code](https://claude.com/code) and [Claude Cowork](https://claude.com/cowork).

## Plugins

### [astro-skill](./astro-skill/) `v1.0.0`

Complete Astro 5.x on Cloudflare Workers/Pages skill. Covers rendering modes (SSG, SSR, hybrid, Server Islands), Content Layer, Cloudflare bindings (KV, D1, R2, Durable Objects), and Astro 5 breaking-change prevention. Includes 3 slash commands for scaffolding, auditing, and debugging.

## Installation

```bash
# Add the marketplace
/plugin marketplace add sebc-dev/marketplace

# Install the plugin
/plugin install astro-skill@sebc-dev-marketplace
```
