## Installation CLI

<github_cli>
| OS | Commande |
|----|----------|
| macOS | `brew install gh` |
| Debian/Ubuntu | `sudo apt install gh` |
| Fedora/RHEL | `sudo dnf install gh` |
| Windows | `winget install GitHub.cli` |

Authentification : `gh auth login`
</github_cli>
